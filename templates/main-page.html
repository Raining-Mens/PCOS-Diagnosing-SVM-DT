HOW TO GUIDE

1. Download or follow the Input Data Excel Template. Download for PCOS or Download for Ovarian Cancer
2. Choose which disease to predict: PCOS or Ovarian Cancer.
3. Select which model to use. It's either Support Vector Machine (SVM) or Decision Tree (DT).
4. Upload the excel file to upload box.
5. Click start to begin evaluation.

It will then show the contents of the file.
